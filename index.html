Welcome to the Patient Orientation Board

Why make the POB an essential part of each hospital room you might ask? Because it is full of unique and useful features such as:

digital display clock

It helps the patients know the current time. And it is very helpful for time specific events for nurses.

Gives a sense of welcoming

The board offers greeting feature that changes based on the time of the day or night; automatically updating starting from 6am displaying "Good morning!".

There is a current date display feature

No need to wipe whiteboards every morning to update the date, the board automatically displays the current date for the patient.

Has a location indicator

The Board displays the location for the patient. This is very helpful for patients who might be confused.

The board automatically displays who is the nurse for the shift

no need to handwrite the nurse's name every shift anymore. The board is connected to the electronic nurses assignment sheet and automatically updates the name of the nurse. So in case of unexpected leave during the shift, no need to update all of the white boards. The board is sensitive to changes made in the assignment sheet. No more confusion!!

Weather forcast feature

The board displayes the current weather outside the building by importing the information from the local weather forcast services.

Keeping up-to-date

There is a local news bar connected to the local news chanel and displays relevant news.
